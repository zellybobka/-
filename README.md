# Компьютерное структурно-функциональное моделирование ВОЛС и оценка качества

Веб-приложение на `Python` и `Streamlit` для построения базовой структурно-функциональной модели волоконно-оптической линии связи, сравнения реальных моделей ВОЛС и расчета итогового рейтинга качества.

## Назначение

Программа предназначена для:

- отображения базовой «идеальной» модели ВОЛС;
- работы с реальными моделями ВОЛС;
- добавления пользовательской модели в расчет;
- формирования матриц `BQ`, `RQ`, `CQ` и весовых коэффициентов `q_l`;
- проверки соответствия модели требованиям;
- расчета комплексного показателя качества эксплуатации `Qкэ`;
- построения итогового рейтинга моделей;
- выгрузки текстового отчета по результатам расчета.

## Используемые технологии

- `Python 3.10+`
- `Streamlit`
- `Pandas`
- `NumPy`

## Важно

Архив содержит исходную поставку программы для `Windows` и `Linux`.

Для запуска программы должны быть предварительно установлены:

- `Python 3.10` или новее;
- доступ к установке зависимостей из файла `requirements.txt`.

После запуска приложение открывается в браузере по локальному адресу вида `http://localhost:8501`.

## Запуск в Windows

1. Установить `Python 3.10` или новее.
2. Распаковать архив программы.
3. Запустить файл `install_windows.bat`.
4. После завершения установки запустить `run_windows.bat`.
5. Дождаться запуска `Streamlit` и открыть адрес `http://localhost:8501`, если браузер не открылся автоматически.

## Запуск в Linux

1. Установить `Python 3.10` или новее.
2. Распаковать архив программы.
3. Выполнить команду `chmod +x install_linux.sh run_linux.sh`.
4. Запустить `./install_linux.sh`.
5. После завершения установки запустить `./run_linux.sh`.
6. Открыть в браузере адрес `http://localhost:8501`.

## Состав архива

- `vols_quality_app.py` - основное веб-приложение `Streamlit`;
- `vols_sfm_app.py` - совместимый лаунчер приложения;
- `requirements.txt` - зависимости проекта;
- `sample_model.json` - пример данных;
- `install_windows.bat` - установка зависимостей и подготовка окружения в `Windows`;
- `run_windows.bat` - запуск приложения в `Windows`;
- `install_linux.sh` - установка зависимостей и подготовка окружения в `Linux`;
- `run_linux.sh` - запуск приложения в `Linux`;
- `README.md` - инструкция по установке и запуску.

## Результат работы программы

Программа позволяет сравнить базовую модель ВОЛС с выбранными реальными моделями, выявить несоответствия по показателям качества, рассчитать `Qкэ` для допустимых моделей и сформировать итоговый рейтинг вариантов построения ВОЛС.
