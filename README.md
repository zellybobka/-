# Базовая СФМ ВОЛС

Кроссплатформенная настольная программа на Python/Tkinter для составления базовой структурно-функциональной модели волоконно-оптической линии связи в табличной форме.

## Назначение

Программа предназначена для формирования базовой СФМ ВОЛС, редактирования строк таблицы, сохранения модели и экспорта данных для дальнейшего использования в отчетных материалах.

## Возможности

- формирование таблицы базовой СФМ ВОЛС;
- добавление, редактирование и удаление строк;
- загрузка и сохранение модели в формате `JSON`;
- экспорт готовой таблицы в `CSV`;
- использование встроенного шаблона для быстрого заполнения.

## Важно

Архив содержит исходную поставку программы для `Windows` и `Linux`. Для запуска программы на компьютере пользователя должны быть предварительно установлены:

- `Python 3.10` или новее;
- для `Linux` также требуется пакет `python3-tk` или аналогичный пакет с библиотекой `Tkinter`.

Программа использует только стандартные модули Python и не требует сторонних библиотек.

## Запуск в Windows

1. Установить `Python 3.10` или новее.
2. Распаковать архив программы.
3. Запустить файл `install_windows.bat`.
4. После завершения установки запустить `run_windows.bat`.

## Запуск в Linux

1. Установить `Python 3.10` или новее.
2. Установить пакет `python3-tk`.
3. Распаковать архив программы.
4. Выполнить команду `chmod +x install_linux.sh run_linux.sh`.
5. Запустить `./install_linux.sh`.
6. После завершения установки запустить `./run_linux.sh`.

## Состав архива

- `vols_sfm_app.py` - исходный код программы;
- `requirements.txt` - файл зависимостей;
- `sample_model.json` - пример заполненной модели;
- `install_windows.bat` - подготовка виртуального окружения в `Windows`;
- `run_windows.bat` - запуск программы в `Windows`;
- `install_linux.sh` - подготовка виртуального окружения в `Linux`;
- `run_linux.sh` - запуск программы в `Linux`;
- `README.md` - инструкция по скачиванию, установке и запуску.
